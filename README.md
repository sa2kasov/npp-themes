Notepad++ Themes
===============================

## Installation:
Copy the xml-files to "%APPDATA%\Notepad++\themes" and in a portable installation to "%Notepad++%\themes".

Or simply import it to Notepad++ by going to Menu -> Settings -> Import -> Import Style theme(s) option.
